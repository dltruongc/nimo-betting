# Introduce

Rảnh quá ngồi viết cái app để bet kèo Nimo.

# Usage

Import source code: tham khảo [Electron Forge](https://www.electronforge.io/import-existing-project).

### Chạy app:

```sh
yarn install
yarn watch
```

### Build app (chỉ mới test trên MacOS):

> Build nằm trong thư mục out

```sh
yarn make
```
